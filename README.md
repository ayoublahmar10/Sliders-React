# Sliders-React


## clone project
$ git clone https://github.com/ayoublahmar10/Sliders-React.git


## install dependencies 
$ yarn install

## build app
$ yarn run build

## run the application with the watch mode 

$ yarn dev 


## run the application
$ yarn start

# http://localhost:3000/ --> will take you directly to slide 0
